# Distribution Observations

## Scope

This file records platform-level access, audience, and engagement observations for the Ω Library and related public entry points.

These observations are not validation evidence for the Ω framework. They do not imply independent reproduction, endorsement, theoretical acceptance, or Colab execution unless such actions are directly reported.

## Measurement policy

- Treat platform screenshots as approximate observations.
- Use observations to improve distribution, navigation, and entry-point design.
- Do not use platform analytics as evidence for Ω validity.
- Do not infer reader intent from views alone.
- Do not infer cross-platform conversion unless directly measured.
- Keep reader-facing entry points separate from maintenance observations.

## Platform roles

- Zenodo: canonical records, archived PDFs, reproducibility packages, DOI-based public reference.
- GitHub / GitHub Pages: runnable entry points, issue routing, reproducibility workflow, navigation hub.
- note: Japanese-language explanatory and distribution channel.
- Substack: English-language mirror / summary channel.
- LinkedIn: professional discovery, targeted specialist reach, connection-based distribution.
- WordPress: stable public web hub.
